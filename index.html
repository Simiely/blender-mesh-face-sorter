大场景里找模型，
大海捞针。

Blender 的大纲视图只能按名称排序。场景里有几百个网格体时，想知道哪个面数最高、哪个最占内存——只能一个一个点开看。与其人肉翻找，不如让插件替你排好：刷新一下，最高面数的模型已经在第一行。

01 / How

一次刷新，全部排好。

STEP 01
📋
刷新列表
扫描场景中所有网格体，收集面数、顶点数、边数、存储大小。大场景显示进度条。
STEP 02
↕️
多维度排序
按面数 / 顶点数 / 存储大小排序，升降序切换。切换排序方式不重扫，瞬时响应。
STEP 03
🎯
定位与减面
点击名称选中物体，一键添加 Decimate 修改器。自定义保留比例，批量或单个操作。
STEP 04
📄
导出报表
一键导出 Markdown 表格，包含统计信息和每物体详情，方便存档或分享。
刷新列表 → 排序切换 → 点击选中 → 减面操作 → 导出报表
02 / Features

三个核心能力。

01

多维度排序

面数、顶点、存储大小三种排序方式，升降序自由切换。切换排序不重扫场景，缓存机制保证瞬时响应。

02

减面工作流

可调保留比例，批量或单个物体添加 Decimate 修改器。支持一键应用修改器，配合清理未使用数据，完整减面闭环。

03

场景管理

孤立显示、删除空网格、清理未使用数据、导出 Markdown 报表。不止排序，是轻量级的场景管理工具箱。

03 / Architecture

缓存优先，手动刷新。
