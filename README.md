# Mesh Face Sorter

> Blender 插件：按面数/顶点/三角面排列场景中的所有网格体，帮助你快速定位高面数模型、批量减面、清理场景。

![License](https://img.shields.io/badge/license-MIT-lightgrey)

## 功能一览

| 功能 | 说明 |
|---|---|
| **面数排序** | 按面数/顶点/三角面 从高到低（或从低到高）排列所有网格体 |
| **一键简面** | 给选中物体或单个物体添加 Decimate 修改器（Collapse 模式，50% 面数） |
| **孤立显示** | 一键隐藏其他所有网格体，只看当前选中的 |
| **删除空网格** | 清理场景中面数为 0 的空网格体（带确认对话框） |
| **导出 md 报表** | 将当前列表导出为 Markdown 表格，含统计信息和每物体详情 |
| **扫描进度** | 扫描大场景时显示百分比进度条，扫描中禁用其他操作 |

## 安装

### 方式一：下载安装（推荐）

1. 访问 [GitHub 仓库](https://github.com/Simiely/mesh-face-sorter)
2. 下载 `mesh_face_sorter.py` 文件
3. 打开 Blender → `编辑 (Edit)` → `偏好设置 (Preferences)` → `插件 (Add-ons)`
4. 点击右上角 `安装 (Install)`，选择下载的 `mesh_face_sorter.py`
5. 在插件列表中搜索 `Mesh Face Sorter`，勾选启用

### 方式二：克隆仓库

```bash
git clone https://github.com/Simiely/mesh-face-sorter.git
```

将 `mesh_face_sorter.py` 按方式一的步骤 3-5 安装即可。

## 使用方法

### 打开面板

1. 在 3D 视图中按 `N` 键打开右侧侧边栏
2. 找到 **`Mesh Sorter`** 标签页

### 面板布局

```
┌─────────────────────────────────────┐
│ 状态区（扫描进度 / 完成时间戳）       │
├─────────────────────────────────────┤
│ 统计区                              │
│   网格体数量：12                     │
│   总面数：1.2M                       │
│   总三角面：2.4M                     │
│   总顶点：600K                       │
├─────────────────────────────────────┤
│ 排序方式：[面数 ▼] [降序 ↑↓]        │
├─────────────────────────────────────┤
│ [刷新列表] [选中所有网格体]          │
│ [显示全部] [删除无面网格体]          │
│ [导出 md 报表]                      │
├─────────────────────────────────────┤
│ [   一键添加简面修改器   ]           │
├─────────────────────────────────────┤
│ # │ 物体名称    │ 面数  │ 👁 │ ▽   │
│ 1 │ Body_High  │ 500K  │ 👁 │ ▽   │
│ 2 │ Head       │ 120K  │ 👁 │ ▽   │
│ 3 │ Hand_L     │ 45K   │ 👁 │ ▽   │
│ ...                                 │
└─────────────────────────────────────┘
```

### 操作说明

| 操作 | 效果 |
|---|---|
| 点击物体名称 | 选中该物体并设为活动对象 |
| 点击 👁 图标 | 孤立显示该物体（隐藏其他所有网格体） |
| 点击 ▽ 图标 | 给该物体单独添加 Decimate 修改器 |
| 点击「刷新列表」 | 重新扫描场景（带进度提示） |
| 点击「一键添加简面修改器」 | 给当前选中的所有网格体批量添加 Decimate |
| 切换排序下拉框 | 按面数/顶点/三角面排序（瞬时切换，不重新扫描） |
| 点击升降序按钮 | 切换降序/升序 |

### 典型使用场景

**场景 1：找出最占资源的模型**
1. 打开面板，点「刷新列表」扫描场景
2. 列表第 1 名就是面数最高的物体
3. 点击名称选中它，决定是否减面

**场景 2：批量减面**
1. 选中多个高面数物体（Shift 多选）
2. 点击「一键添加简面修改器」
3. 所有选中物体都会加上 Decimate（50% 面数）
4. 在修改器属性里调整 ratio 滑块精细控制

**场景 3：导出场景报告**
1. 点「导出 md 报表」
2. 选择保存路径
3. 得到包含统计信息和完整列表的 Markdown 文件

**场景 4：清理脏数据**
1. 点「删除无面网格体」
2. 确认对话框显示数量
3. 确认后清空所有空网格体

## 性能说明

插件针对大场景做了专门优化：

- **缓存机制**：Panel 不会每帧重新扫描，避免卡顿
- **手动刷新**：只有点「刷新」按钮或加载新文件时才扫描
- **C API 加速**：三角面数用 `mesh.loop_triangles`（C 实现），比 Python 循环快几十倍
- **列表限制**：最多显示 500 个物体，超出可导出报表查看全部
- **进度提示**：扫描时显示百分比，扫描中禁用其他操作

> 加载大型 FBX（几千个网格体）后也能流畅使用。

## 常见问题

**Q：安装后找不到插件？**
A：偏好设置搜索框输入 `Mesh Sorter` 过滤。

**Q：列表没更新？**
A：点「刷新列表」按钮重新扫描（手动刷新模式）。

**Q：物体没出现在列表？**
A：只统计 `type == 'MESH'` 的物体，曲线/灯光/相机不会出现。

**Q：面数和属性面板不一致？**
A：列表显示的是多边形面数（`mesh.polygons`），Blender 状态栏有时显示三角面数，切换到「三角面」排序即可对应。

**Q：扫描时界面卡住？**
A：扫描是同步操作，几千个物体时需等几秒，进度条会在扫描完成后显示。期间不要操作 Blender。
